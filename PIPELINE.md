# InSIGHT 파이프라인 설명

## 전체 흐름

```
입력 (Instagram URL / 이미지 파일)
        ↓
   이미지 획득
        ↓
┌─────────────────────────────┐
│         1차 필터             │  ← 결정론적 메타데이터 기반 (빠름, 무료)
│  C2PA → EXIF → SynthID/FFT │
└─────────────────────────────┘
        ↓ 감지됨              ↓ 감지 안 됨
   ❌ AI 생성 확인     ┌─────────────────────────────┐
   (종료)             │         2차 필터             │  ← AI 모델 기반 (정밀)
                      │     Gemini 2.0 Flash         │
                      └─────────────────────────────┘
                               ↓
                  ❌ AI 생성 / ✅ 실제 / ❓ 불확실
```

---

## 입력 단계

| 입력 방식 | 처리 방법 |
|-----------|-----------|
| Instagram URL | instaloader로 shortcode 추출 → 이미지 URL 획득 → 다운로드 |
| 일반 이미지 URL | requests로 직접 다운로드 |
| 파일 업로드 | PIL로 직접 로드 |

> Instagram은 로그인 없는 크롤링을 차단하는 경우가 있어 파일 업로드를 권장합니다.

---

## 1차 필터 — 메타데이터 기반

결정론적으로 작동하며 GPU 없이 빠르게 실행됩니다. 하나라도 양성이면 2차 필터를 건너뜁니다.

### C2PA 메타데이터 검증
- **방법:** `c2pa-python` 라이브러리로 이미지의 Content Credentials 매니페스트 파싱
- **판단 기준:** 매니페스트 내 `generativeai`, `dall`, `midjourney`, `stable diffusion`, `firefly`, `imagen`, `ai.generated` 등 키워드 탐지
- **한계:** C2PA 서명이 없는 이미지(대부분의 SNS 이미지)는 탐지 불가

### EXIF 메타데이터 분석
- **방법:** PIL로 EXIF 태그 전체 추출 후 AI 도구 키워드 검색
- **판단 기준:**
  - AI 도구 흔적 발견 시 → 양성
  - 카메라 제조사/모델 정보 존재 시 → 음성 (실제 카메라 촬영 가능성)
  - EXIF 없음 → 불확실 (AI 이미지는 EXIF가 없는 경우가 많음)
- **탐지 키워드:** `dall-e`, `midjourney`, `stable diffusion`, `adobe firefly`, `imagen`, `synthid` 등

### SynthID / FFT 주파수 분석
- **방법:** 이미지를 그레이스케일 변환 후 FFT(고속 푸리에 변환) 적용, 고주파 영역의 피크 비율 측정
- **판단 기준:** `피크값 / 평균값 > 1.8` 이면 비정상 주파수 패턴으로 판정
- **원리:** AI 워터마크(SynthID 등)는 특정 주파수 대역에서 비정상적으로 강한 피크를 남김
- **한계:** 근사적 탐지이며 실제 SynthID 검출기와 동일하지 않음

## 2차 필터 — AI 모델 기반

1차 필터를 통과한 이미지에 대해 Gemini 멀티모달 모델로 시각적 분석을 수행합니다.

### Gemini 2.0 Flash (Google)
- **입력:** 이미지 + 분석 프롬프트
- **분석 항목:**
  1. 피부·텍스처의 과도한 매끄러움
  2. 손가락, 귀, 치아 등 세부 부위 형태 이상
  3. 배경의 비논리적 구조 또는 반복 패턴
  4. 조명·그림자의 물리적 불일치
  5. 눈동자 반사 또는 홍채 비현실성
  6. 텍스트·문자의 왜곡
- **출력 형식:**
  ```
  판정: AI 생성 / 실제 이미지 / 불확실
  신뢰도: 0~100%
  근거: (한국어 2~3문장)
  ```

## 최종 판정 기준

| 결과 | 조건 |
|------|------|
| ❌ AI 생성 이미지 (1차 필터) | C2PA / EXIF / FFT 중 하나라도 양성 |
| ❌ AI 생성 이미지 (2차 필터) | Gemini가 "AI 생성"으로 판정 |
| ✅ 실제 이미지 | Gemini가 "실제 이미지"로 판정 |
| ❓ 불확실 | Gemini가 "불확실"로 판정 또는 오류 |

> 워터마크 **감지됨** → "AI 생성 확실"  
> 워터마크 **없음** → "AI 아님"이 아닌 **"증거 없음"**  
> 이 차이를 반드시 유의해야 합니다.

|------|------|
